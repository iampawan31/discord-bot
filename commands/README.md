# Simple Discord Chatbot using NodeJs

## Requirements

- [NodeJS](https://nodejs.org)
- [Discord](https://discordapp.com) Account

## Installation Steps

1. Clone Repo
2. Run `npm install`
3. Add Discord Credentials (Bot `TOKEN`) in `.env` file
4. Run `node index.js`
5. Interact with your Discord bot via browser/app
